Descripción del Proyecto

Este proyecto corresponde a la asignatura ISW-306 y tiene como objetivo desarrollar una aplicación web profesional de manera incremental. Se han implementado estructuras semánticas con HTML5, estilos mediante CSS y funcionalidades interactivas con JavaScript para la validación de formularios y la persistencia temporal de datos mediante LocalStorage.

Etapa 1: Estructura Semántica y Estilización

* Uso de HTML5 semántico.
* Aplicación de estilos con CSS3.
* Uso de colores institucionales.
* Diseño responsive.

Etapa 2: Interactividad y Persistencia

* Validación de formularios en el cliente.
* Manipulación del DOM.
* Mensajes dinámicos de error y éxito.
* Uso de expresiones regulares para validar correos electrónicos.
* Almacenamiento de datos mediante LocalStorage.
* Recuperación automática de los datos al recargar la página.

Tecnologías Utilizadas

* HTML5
* CSS3
* JavaScript
* Git
* GitHub
* Visual Studio Code

Repositorio

El desarrollo del proyecto se realiza mediante control de versiones con Git y GitHub, utilizando ramas independientes para cada etapa del proyecto.
