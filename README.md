# Proyecto Integrador ISW-306

## Etapa 3: Backend y Persistencia de Datos

### Descripción
Este proyecto implementa el backend del sistema utilizando ASP.NET Core y SQL Server. Incluye una API REST para la gestión de usuarios y la persistencia de datos mediante Entity Framework Core.

---

## Tecnologías utilizadas
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- Swagger

---

## Funcionalidades
- Crear usuarios (POST)
- Consultar usuarios (GET)
- Actualizar usuarios (PUT)
- Eliminar usuarios (DELETE)

---

## Base de datos
Se utiliza SQL Server con migraciones automáticas de Entity Framework Core.
