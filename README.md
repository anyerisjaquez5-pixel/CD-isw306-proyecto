Descripción del Proyecto

Este proyecto corresponde a la asignatura ISW-306 y tiene como objetivo desarrollar una aplicación web profesional de manera incremental. En esta primera etapa se implementó la estructura semántica utilizando HTML5 y CSS, aplicando los colores institucionales y técnicas de diseño responsive.

Tecnologías Utilizadas

* HTML5
* CSS3
* Git
* GitHub
* Visual Studio Code
